HTML

HTML is the standard markup language for creating web pages. It provides the basic structure of a webpage, allowing developers to define various elements such as headings, paragraphs, images, links, and more.

CSS

CSS is a stylesheet language used to describe the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance of web pages.

PHP

PHP is a server-side scripting language primarily used for web development. It allows developers to create dynamic content that interacts with databases.
